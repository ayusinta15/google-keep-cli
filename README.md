# Google Keep CLI
Aplikasi clone Google Keep berbasis Command Line Interface (CLI) menggunakan bahasa PHP.

## Deskripsi
Google Keep CLI merupakan aplikasi pencatatan sederhana yang memungkinkan pengguna untuk membuat dan mengelola catatan melalui terminal. Project ini dibuat dengan menerapkan konsep Object Oriented Programming, Repository Pattern, Service Layer, serta Unit Testing.

## Menjalankan Aplikasi
Aplikasi Google Keep CLI dapat dijalankan melalui terminal menggunakan perintah:
- php index.php

## Fitur
* Menambahkan note
* Melihat semua note
* Mengubah note
* Menghapus note
* Mencari note
* Pin note
* Melihat archive note

## Teknologi yang Digunakan
* PHP 8.2.29
* Composer
* MySQL
* PHPUnit
* Mockery
* Xdebug


## Testing
Project ini menggunakan PHPUnit untuk unit testing dan Mockery untuk mocking.
Untuk menjalankan testing:
- vendor/bin/phpunit

Dokumentasi lengkap mengenai testing, mocking, dan code coverage dapat dilihat pada:
- docs/TESTING.md
